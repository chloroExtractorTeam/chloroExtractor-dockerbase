# chloroExtractor-dockerbase
Docker base image for our chloroExtractor Docker container

This image provides the base installation for the chloroExtractor
docker image.

## Installed software
### Perl modules
* Log::Log4perl
* Moose
* File::Which
* IPC::Run
* Term::ProgressBar
* Graph
### Software
* bowtie2
* NCBI-blast
* samtools
* GNU R
* Ghostscript
* jellyfish
* Python
* SPAdes
